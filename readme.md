# Исторические даты - Тестовое задание

Визуализация исторических событий на соответствующих временных сегментах.  
Технологии: React, TypeScript, SCSS

---

## Функциональность

- **Сегменты времени**: сегменты с заданными годами и метками (технологии, кино, литература и т.д.).  
- **Круг с сегментами**: визуальное расположение сегментов на окружности. Активный сегмент всегда в первой четверти. Круг поворачивается анимацией при переключении сегментов.  
- **Переключение сегментов**:  
  - Стрелками навигации
  - Клик по точкам на круге (на большом экране)
  - Пагинация (на маленьком экране)  
- **События внутри сегмента**:  
  - Листаются с помощью Swiper  
  - Фильтруются по годам и меткам  
- **Годы** начала и конца активного сегмента.  

---

## Установка и запуск проекта

Клонируйте репозиторий:  
git clone https://github.com/Ekvisa/HistoricalEvents  
Установите зависимости:  
npm install  
Запустите проект:  
npm run dev  
Проект запустится на локальном сервере.  

## Структура проекта  
src/components/App — главный компонент, управляет активным сегментом  
src/components/Circle — вращающийся круг с переключателями сегментов  
src/components/Events — список событий внутри сегмента  
src/components/Years — отображение диапазона лет сегмента  
src/types/ и src/data/ — данные и типы
